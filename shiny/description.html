Instructions

  • Select the measures and countries you wish to visualise, including trends in inequality, homicide rates, and financial indices.
  • This visualization allows comparisons of selected measures over time.
  • You can also overlay world events by checking the 'Show World Events' box.
  • Click on the 'Randomly Select Parameters'-Button to get a random selection of parameters with respect to the selected country/countries.
  • The 'Download as .csv'-Button allows to download the data that is selected as csv-file.

Note

Please note that not all countries have data for all measures. If a country is selected that does not have data for a measure, the graph will not display data for that measure. Furthermore, some countries have only limited data, so lines may not be displayed for the entire time range.
